16. **Задача** Да се състави функция, която по цяло число **n** определя дали в двоичното му представяне няма повтарящи се съседни битове.

	**Пример**<br>
	Вход:
	```text
	85
	```
	Изход:
	```text
	true
	```

	---

	Вход:
	```text
	93
	```
	Изход:
	```text
	false
	```
	---
	Обяснение на примерите:
	```
	85 → 1010101
	93 → 1011101
	```
